# TUST-ACM战队 算法入队选拔机制
因为经费有限&&xcpc本身有名额限制，所以不会所有人都去打xcpc（蓝桥杯等没有名额限制的竞赛不再讨论范围内），所以我们会有内部的算法入队选拔赛：

11-1月份是队内集训，集训后我们会有8-12场左右的线上竞赛，这些线上竞赛可能是codeforces，atcoder，力扣，牛客竞赛，蓝桥竞赛等线上比赛；每场比赛我们会统计参赛队员的得分，得分规则如下：

| 比赛名次 | 得分 | 备注 |
|----------|----------|----------|
| 1    | 20     | 无     |
| 2    | 15     | 无     |
| 3    | 12     | 无     |
| 4    | 10     | 无     |
| 5    | 8     | 无     |
| 6    | 7     | 无     |
| 7    | 6     | 无     |
| 8    | 5     | 无     |
| 9    | 4     | 无     |
| 10    | 3     | 无     |
| >10    | 1     | 无     |
| 未参加比赛    | 0     | 无     |

那么，8-12场比赛后，每个人都会有一个自己的得分；最终我们会产出一个队内每个人的得分，得分作用是建队，具体建队规则如下：
1. 总得分第一名优先挑选自己的两名队友并和这两人协商，如果这两人同意组队（例如第一名挑选了第3名和第5名作为队友&&第三名和第五名均同意），那么这个队伍就会作为TUST的算法一队。
2. 接着，看下一个没有成立队伍的最高名次，此时是第二名，那么，第二名优先选择自己的两位队友并征求两名队友的同意后成立TUST的算法二队
3. 第一届算法队伍不超过4个队伍（如果有女队，那么女队另算）
4. 剩下没有入队的同学会分流去张老师的项目组进行项目学习和参与；但是并非这类同学就没有机会回来打xcpc，未来这类同学如果要回来打xcpc有两个途径，第一cf/atc分达到TUST二队的第一名的水平，第二如果经费充足那么可能可以参加ccpc的省赛
5. 以上规则仅适用xcpc

## 运行机制
每周都会有一个小组作为负责人，负责人由3人一组组成，一个人最多当2次负责人；大家之间互相服务
### 负责人工作内容
#### 提前学习负责该周的内容
#### 选题单
每周作业题单题量在10-20题（取决于题目难度，如果偏难题目数量就少一点，不要脱离大部队的实力选题，比如选cf2000分的题，没有任何意义），题单选择可参考下面：
https://github.com/EndlessCheng/codeforces-go/blob/master/leetcode/SOLUTIONS.md
https://github.com/Lightblues/Leetcode
题单不用局限于什么平台，力扣，cf，atc等等都无所谓，关键是保证题目质量
#### 3人互相验题
举个例子，比如hangz,zzh,ws三人一组是第12周的负责人，那么至少提前一周，也就是第11周要准备好第12周的题目，并且3人之间都把选出来的12-20题全部做完，保证题目质量

# 时间轴timeline
1. 11-1月三个月，集中训练，夯实基础
2. 2月-3月，寒假集训&队内赛，统一在实验室参加8-12场线上竞赛，每个人取得个人积分
3. 根据个人积分分出1～4队，没进队同学分流进项目组（没进队可以自己继续训练算法）
4. 4月蓝桥杯，全员参与，后续所有算法竞赛如果要填指导老师都要填上张传雷教授
5. 5月-6月ccpc省赛，暂定至少参加河北河南两场省赛
6. 7-8月暑假集训，准备icpc
7. 9月开学的前三周(一般这样)，每个周末一把网络预选赛，争取icpc学校的参赛资格，要求全员参与
